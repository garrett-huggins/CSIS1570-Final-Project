How it's built.

Godot

Built with the latest version of the Godot engine, v3.3

Audacity

Music and sound are created using the latest version of Audacity, v3.1.2

Aseprite

Art assets are designed using the latest version of Asesprite, v1.3

What is Godot?

Godot is a free and open-source game engine that supports 2D and 3D game development.

Godot supports C# and even a redimentary visual scripting language, as well as using its own language GD Script.
